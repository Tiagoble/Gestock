<h2>Objetivo</h2>
O Gestock é um sistema de gestão de vendas que foi criado para facilitar as 
vendas e a gestão do estoque, otimizando o trabalho dos usuários e mantendo a 
organização dos produtos e clientes.

<br>
<h2>Funcionalidades</h2>
Cadastro e Consulta de produtos<br>
Atualizar e excluir produtos<br>
Registrar e atualizar vendas<br>
Cadastrar, Editar e Excluir clientes<br>
